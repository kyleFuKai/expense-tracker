# 修改底部加号按钮布局

## Context
当前 [MainLayout.vue](finance-app/src/layouts/MainLayout.vue) 底部栏有 4 个 item（首页、+、统计、我的），加号在第 2 位，`space-around` 布局下**没有居中**，且 `margin-top: -20px` 让它浮出栏外，视觉别扭。改为 3 个 tab + 底部栏右侧悬浮 FAB（Material Design 风格）。

## 改动

**仅改一个文件：** `finance-app/src/layouts/MainLayout.vue`

### 模板
- 去掉 `tab-add` 项（`<div class="tab-add">`），tab 栏只剩 3 个 `router-link`：首页 / 统计 / 我的
- 在 `.tab-bar` 内部末尾添加 `<div class="fab-add" @click="openBillForm">`（绝对定位，在底部栏右侧凸起）

### 样式
- `.main-layout`：`padding-bottom` 从 60px 增加到 84px（56px tab + FAB 凸出部分）
- `.tab-bar`：保持 `position: fixed; bottom: 0` 不变，新增 `overflow: visible`（确保 FAB 不被裁剪）
- `.tab-bar`：`justify-content: space-evenly`（3 个 tab 均匀分布）
- `.tab-item`：保持不变，`flex: 1`
- 去掉 `.tab-add` 及其所有样式
- 新增 `.fab-add`：
  ```css
  .fab-add {
    position: absolute;
    bottom: calc(100% - 24px);  /* 圆直径48，一半24，让圆一半骑在栏上 */
    right: 16px;                 /* 距右边缘16px */
    width: 48px; height: 48px;
    border-radius: 50%;
    background: var(--color-primary);
    display: flex; align-items: center; justify-content: center;
    box-shadow: 0 4px 12px rgba(60, 197, 92, 0.4);
    cursor: pointer;
    z-index: 10;
  }
  .fab-add :deep(.van-icon) { font-size: 24px; color: #fff; }
  ```
  这样 FAB 一半在栏上、一半在栏外（右侧），视觉效果好，与随手记/鲨鱼记账等竞品一致

## 验证
- `npx vue-tsc --noEmit` 类型检查通过
- dev server 启动后确认：
  - 底部 3 个 tab 均匀分布
  - FAB 在底部栏右侧凸起
  - 点击 FAB 能打开记账表单
