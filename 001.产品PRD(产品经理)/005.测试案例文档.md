## 一、测试概述

### 1.1 项目简介

每日财务管家是一款个人/家庭财务管理应用，主要功能包括：
- 用户认证（注册、登录、密码管理）
- 账单管理（收入/支出记录、分类、标签）
- 预算管理（总预算、分类预算）
- 统计分析（月度统计、分类排行、支出趋势）

### 1.2 技术栈

| 层级 | 技术 |
|------|------|
| 前端 | 原生 JavaScript + Tailwind CSS 3 (CDN) + Material Symbols |
| 后端 | Node.js v22.20.0 + Express 4.x |
| 数据库 | MySQL 8.0 (utf8mb4, dateStrings: true) |
| 认证 | JWT (7 天有效期) + BCrypt 密码哈希（spring-security-crypto） |

### 1.3 测试目标

- 验证 Java 后端 Service 层逻辑正确性（JUnit 5）
- 验证所有 API 接口功能正确性（MockMvc 集成测试）
- 验证边界条件和异常处理
- 验证认证和授权机制
- 验证数据一致性和完整性

## 八、附录

### 8.1 API 响应格式标准

```json
{
  "code": 0,        // 0=成功, 非 0=错误码
  "message": "ok",  // 响应消息
  "data": {}        // 响应数据
}
```

### 8.2 常见错误码

| Code | 说明 |
|------|------|
| 200 | 成功 |
| 400 | 请求参数错误 |
| 401 | 未认证/Token 无效 |
| 403 | 无权限 |
| 404 | 资源不存在 |
| 409 | 资源冲突（如重复注册） |
| 500 | 服务器内部错误 |
