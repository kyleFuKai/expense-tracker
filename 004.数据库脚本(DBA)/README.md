目录结构:
├── 001_schema_ddl.sql      -- 建表脚本 (7 张表)
├── 002_seed_data.sql       -- 预设分类数据 + 测试用户
└── 003_er_model.sql        -- ER 关系图说明

执行顺序:
  1. 001_schema_ddl.sql   (先建表)
  2. 002_seed_data.sql    (再导入预设数据)

数据库要求:
  - MySQL 8.0+
  - 字符集: utf8mb4
  - 排序规则: utf8mb4_unicode_ci

表清单:
  V1.0 核心表:
    - user                       用户表
    - user_third_party_account   第三方账号绑定表
    - category                   收支分类表
    - bill                       账单记录表
    - bill_tag                   账单标签表
    - bill_tag_rel               账单-标签关联表
  V1.1 扩展表:
    - budget                     预算表

注意事项:
  1. 001_schema_ddl.sql 包含 IF NOT EXISTS，可重复执行
  2. 002_seed_data.sql 为 INSERT 语句，重复执行会插入重复数据
  3. 生产环境请修改测试账号密码
  4. 预设分类使用 INSERT 不加 IF NOT EXISTS，首次导入即可
